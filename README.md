

 Task 01 — Landing Page

 Project Description

This project is a responsive and user-friendly landing page created as part of my SkillCraft Technology internship tasks. The landing page is designed to give visitors a clear overview of a product, service, or brand using a modern, clean layout and simple navigation.
 
  Features

Responsive design (works on desktop and mobile)

Attractive layout using HTML and CSS

Smooth navigation sections (Home, About, Services, Contact)

Call-to-action buttons

Organized and easy-to-read structure


 Technologies Used

HTML5 – structure of the webpage

CSS3 – styling and layout

(Optional) JavaScript – for interactivity (if you added any animations or dynamic effects)


 Learning Outcome

Through this task, I learned how to:

Build a structured web page using HTML

Style components effectively with CSS

Make a page responsive and visually appealing


 How to Run

1. Download or clone the project folder.


2. Open the file index.html in any web browser.


3. The landing page will open and display all sections.
